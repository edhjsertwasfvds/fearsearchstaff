# Запуск localhost

## 1. Установите Node.js

**Node.js не установлен** — установите с официального сайта:

- **https://nodejs.org/** — скачайте LTS версию
- Запустите установщик, пройдите все шаги
- **Перезапустите терминал** (или IDE) после установки

Проверка установки:
```powershell
node --version
npm --version
```

## 2. Запуск проекта

### Вариант A: Дважды клик по `start.bat`
Просто запустите файл `start.bat` — он установит зависимости (если нужно) и запустит сервер.

### Вариант B: Из терминала
```powershell
npm install
npm start
```

## 3. Откройте в браузере

- **http://localhost:3000** — главная страница
- **Вход:** логин `admin`, пароль `admin123` (создаётся из `.env` при первом запуске)

## Если не работает

- Убедитесь, что порт 3000 свободен
- Проверьте, что файл `.env` существует в корне проекта
- После установки Node.js может потребоваться перезапуск IDE/терминала
