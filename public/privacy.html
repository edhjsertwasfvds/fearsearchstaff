Политика конфиденциальности

FearSearch — это инструмент управления стаффом и проверки игроков для сообщества Counter-Strike 2. Данная политика объясняет, какие данные собираются и как они используются.

Какие данные мы собираем

  • Discord ID — для идентификации пользователя при входе и подтверждении регистрации.
  • ID ролей Discord — для определения уровня доступа на сайте (модератор, администратор, владелец).
  • Steam ID — для связи аккаунта сайта с Discord-аккаунтом при регистрации.
  • Логины и пароли — для аутентификации на сайте. Пароли хешируются с помощью bcrypt.
  • Логи входов (IP-адрес, User-Agent, страна, город, устройство, браузер, ОС) — для безопасности и аудита.
  • Содержимое сообщений — бот может обрабатывать команды и вложения стаффа в Discord-каналах.

Как используются данные

Все данные используются исключительно для управления доступом стаффа к панели сайта и автоматизации проверок. Мы не продаём, не передаём и не публикуем данные третьим лицам.

Хранение данных

Данные хранятся в базе данных PostgreSQL, размещённой на платформе Railway. Данные в состоянии покоя шифруются средствами Railway. Пароли хешируются с использованием bcrypt.

Discord

Бот использует Server Members Intent для чтения ролей стаффа и Message Content Intent для обработки команд и вложений в Discord-каналах. Бот взаимодействует только с членами стаффа.
